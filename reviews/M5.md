## Findings

### 1. [required — robustness/availability] Short secret-named environment values corrupt evaluation

`src/gauntlet/evidence/content.py` (`SecretRedactor.__init__`): redaction literals are harvested from **every** environment variable whose *name* matches the secret pattern, with **no minimum value length** — `if value and (...)`. Any short value is then substring-replaced across all persisted JSON, including schema-critical structural fields.

**Deterministic minimal repro (reviewer-verified):** with `MY_API_KEY=20` set — nothing else — `tests/test_mvp_acceptance.py::test_acceptance_04_sample_agent_evaluation_completes` errors with:

```
Error: Redaction made ScenarioResult invalid: 2 validation errors for ScenarioResult
... datetime_from_date_parsing
```

The literal `"20"` is replaced inside ISO timestamps (`2026-…` → `[REDACTED]26-…`), the post-redaction model validation fails closed, and the whole evaluation crashes. In the reviewer's real container environment this broke 8 of 267 tests and would break `gauntlet evaluate` for any user whose laptop exports a secret-named variable with a short value (`TOKEN=dev`, numeric IDs, etc.) — precisely the target user per `03_PRD.md`.

The fail-closed behavior is correct and must be kept; the defect is that the redactor treats un-secret-like short strings as secrets at all.

**Required fix (architect decision):** exclude environment-harvested literals shorter than a documented minimum — add a module constant (e.g. `SECRET_LITERAL_MIN_LENGTH = 8`) and apply it to all name-pattern-harvested and explicitly named environment values; values below the threshold are never used as redaction literals. Configured regex `patterns` are unaffected. Rationale to record in code/HANDOFF: sub-8-character strings are overwhelmingly coincidental substrings (dates, counters, ports); replacing them corrupts structured artifacts, while genuine credentials are long.

**Required regression tests with the fix:**
1. With a short secret-named variable set (e.g. `MY_API_KEY=20`), a full evaluation completes, timestamps are intact, and `20` appears unredacted where it legitimately occurs.
2. A ≥8-character secret value in the environment is still redacted from every persisted artifact (existing whole-run redaction test may be extended).

Commit as `WP-5.3 fix: <summary>`; re-run the full Windows gate and record it in `HANDOFF.md`. Reviewer will re-run the Linux gate **in an unscrubbed environment** — the suite must pass with the reviewer's real environment variables present.
