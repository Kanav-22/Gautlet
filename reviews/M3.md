## Verification against the M3 authorization (reviews/M2.md)

- **WP-3.1:** executor implements the full documented lifecycle (Loaded → Validated → Prepared → Running → terminal → Finalized) with `timed_out` as a first-class terminal status per ADR-006; timeouts map to the adapter's parent-enforced deadline; retries are bounded, apply only to child errors already marked `retryable`, and always use a fresh child with rewound fixtures; cleanup is unconditional; concurrency is bounded and input-order preserving.
- **WP-3.2:** evidence envelopes are content-addressed (`sha256:` over the exact persisted redacted bytes); **redaction happens in memory before any write** — literals harvested from secret-named environment variables plus configurable regex patterns, fail-closed on redacted-key collisions; tamper detection on read; every result links to its evidence by semantic role.
- **WP-3.3:** all nine MVP assertion types implemented with strict pydantic definitions (discriminated by type, extra keys forbidden); every `AssertionResult` requires non-empty, unique, *verified* evidence references — unknown references are rejected.

### Accepted deviations (documented in HANDOFF.md, all reasonable)

1. Runtime `jsonschema` (+`types-jsonschema` dev) so `schema_valid` is real JSON Schema Draft 2020-12 instead of a custom subset; non-local `$ref` rejected to preserve offline evaluation. Accepted — better than the alternative.
2. `execution_policy.max_retries` (default 0) as the retry-policy key — specs left this undefined; the choice is conservative.
3. Network gate implemented as network-policy environment isolation (no proxy/API-key/secret inheritance), explicitly not claiming OS-level socket denial — consistent with ADR-002's honesty stance.
4. `no_hallucinated_success` = deterministic fixture-consumption proof (a `completed: true` claim must be backed by all declared fixtures consumed in order). Accepted; framework-neutral and evidence-backed.
5. Windows extended-length path handling inside artifact I/O with portable relative evidence paths retained.
